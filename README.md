


This is a simple KDE Dolphin Context Menu Service Action to create alaias (shortcut) from selected file like in Windows.

### Installation
Put `create_alias.desktop` to `/home/deck/.local/share/kio/`<br/>
Allow executing (Right click on file > Properties > Permissions > ckechkbox on Allow executing file as program)<br/>

Tested on SteamOS (Deck)
